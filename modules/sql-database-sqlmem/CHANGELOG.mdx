<Update label="2025/08/19" description="sqldb-sqlmem-v0.1.0">
  - Initial release of the SQL Database SQL Instance Memory Percent alert module.

  - Creates an Azure Monitor metric alert for `sql_instance_memory_percent` with default name format `alrt-sqlmem-<resource-name>`.

  - Configurable threshold, description, enablement, auto-mitigation, and action groups.

  - Outputs include `id` and `name`.

  </Update>
