<Update label="2025/08/19" description="st-used-v0.1.0">
  - Initial release: Storage Account Used Capacity metric alert.

  - Monitors `UsedCapacity` from `Microsoft.Storage/storageAccounts` with default name format `alrt-used-<resource-name>`.

  - Configurable threshold, description, enablement, and action groups.

  - Outputs include `id` and `name`.

  </Update>
