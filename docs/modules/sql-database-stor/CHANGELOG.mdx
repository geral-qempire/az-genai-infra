<Update label="2025/08/19" description="sqldb-stor-v0.1.0">
  - Initial release of the SQL Database Storage Percent alert module.

  - Creates an Azure Monitor metric alert for `storage_percent` with default name format `alrt-stor-<resource-name>`.

  - Configurable threshold, description, enablement, auto-mitigation, and action groups.

  - Outputs include `id` and `name`.

  </Update>
