## 1. Module
Exposes a canonical map of Azure region names to short abbreviations.

## 2. Usage
```hcl
module "region_abbreviations" {
  source = "./modules/region-abbreviations"
}

locals {
  region_abbr = lookup(module.region_abbreviations.regions, var.location, null)
}
```

## 3. Inputs
None.

## 4. Outputs
| Name | Description |
|------|-------------|
| `regions` | Map of Azure location names (both canonical and display formats) to abbreviations. |

## 5. Requirements
- Terraform `>= 1.12.1`
