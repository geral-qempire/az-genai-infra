<Update label="2025/08/09" description="sqldb-avail-v0.1.0">
  - Initial release of the SQL Database Availability alert module.

  - Creates an Azure Monitor metric alert for SQL Database `availability` metric with default name format `alrt-avail-<resource-name>`.

  - Configurable threshold, description, enablement, auto-mitigation, and action groups.

  - Outputs include `id` and `name`.

  </Update>
