## 1. Module
Create an Azure Monitor metric alert for Azure AI Search throttled queries percentage:
- Metric: `ThrottledSearchQueriesPercentage`
- Namespace: `Microsoft.Search/searchServices`
- Aggregation: Average
- Operator: GreaterThan
- Evaluation Frequency: 1 minute (PT1M)
- Time Window: 5 minutes (PT5M)
- Severity: 3
 - Default Threshold: 10
- Severity: 3

## 2. Usage
```hcl main.tf
module "throttled_search_pct_alert" {
  source = "../ai-search-thrpct"

  # Optional name. If omitted, defaults to: alrt-thrpct-<resource-name>
  # name = "alrt-thrpct-mysearch"

  resource_group_name = "rg-observability-prod"
  scopes              = [module.ai_search.search_service_id]

  # Alert when throttled search queries exceed 10% over 5 minutes
  threshold = 10

  # Optional settings
  description   = "Average throttled search queries percentage over 5 minutes above 10%"
  enabled       = true
  auto_mitigate = true

  action_group_ids = [
    module.action_group.action_group_id
  ]

  tags = {
    environment = "prod"
    project     = "acme"
  }
}
```

## 3. Inputs
| Name | Type | Default | Required | Description |
|------|------|---------|:--------:|-------------|
| `name` | `string` | `null` | no | Name of the metric alert. Defaults to `alrt-thrpct-<resource-name>`. |
| `resource_group_name` | `string` | n/a | yes | Resource group in which to create the alert. |
| `scopes` | `list(string)` | n/a | yes | List of Search Service resource IDs to monitor. |
| `threshold` | `number` | `10` | no | Percentage threshold. Fires when average over 5 minutes exceeds this value. |
| `description` | `string` | `null` | no | Custom alert description. Auto-generated if not provided. |
| `enabled` | `bool` | `true` | no | Whether the alert is enabled. |
| `action_group_ids` | `list(string)` | `[]` | no | Action Group IDs to notify when the alert fires. |
| `auto_mitigate` | `bool` | `true` | no | Whether the alert should auto mitigate when conditions clear. |
| `tags` | `map(string)` | `{}` | no | Tags to apply to the metric alert resource. |

## 4. Outputs
| Name | Description |
|------|-------------|
| `id` | Resource ID of the metric alert. |
| `name` | Name of the metric alert. |

## 5. Requirements
- Terraform `>= 1.12.1, < 2.0.0`
- AzureRM provider `~> 4.38`
- Existing Resource Group
- Existing Azure AI Search Service
